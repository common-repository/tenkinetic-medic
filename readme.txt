Get some CSS and JavaScript to fix or enhance your site.

== Description ==

NOTE: This plugin is currently intended for use by tenKinetic clients only.

tenKinetic Medic allows you to get help for issues with your site from an experienced developer. All solutions are delivered to your site
via the plugin and no passwords to access your systems are required. All solutions are CSS and JavaScript based and can be activated
by you for the pages you choose.

The basic process: create a job, submit it to the medic, exchange any messages regarding clarification, get a quote for the work, pay, receive
patches, configure patches, admire your updated page(s), accept job.

These usage instructions are available in more detail within the plugin itself.

== Installation ==

Install and activate the plugin. The plugin will then be available from the administration menu in WordPress.
The plugin will determine if you have the required versions of PHP (5.3.0) and MYSQL (5.6.5).
